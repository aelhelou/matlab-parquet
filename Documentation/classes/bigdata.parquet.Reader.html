bigdata.parquet.Reader
  Class for reading Parquet files
 
  Example: Read in at most 100 rows from a Parquet file
 
    import bigdata.parquet.Reader
    p = Reader('FileName','tmp.parquet','MaxRows',100)
    data = p.read;
